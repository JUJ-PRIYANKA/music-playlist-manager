# Music Playlist Manager

A Spring Boot web application for managing a music playlist.

## Features

* Add songs
* Delete songs
* Shuffle playlist

## Technologies Used

* Java
* Spring Boot
* H2 Database
* HTML
* CSS
* JavaScript

## How to Run the Project

1. Clone the repository
2. Open the project in VS Code or IntelliJ
3. Run `MusicplaylistApplication.java`
4. Open browser and go to:

http://localhost:8080
